# D-day 과제 처리기

## 한 문장 설명
여러 과제의 마감일, 과제 종류, 난이도를 입력하면 남은 날짜와 우선순위를 계산해 먼저 해야 할 과제를 알려주는 프로그램입니다.

## 풀려는 문제 또는 만들려는 이유
저 같은 학생들은 여러 과제와 시험 일정을 동시에 관리해야 해서 마감일을 놓치기 쉽습니다. 또한 마감일이 가까워도 과제의 난이도나 종류에 따라 실제로 필요한 시간이 달라지기 때문에, 매번 어떤 과제를 먼저 해야 할지 직접 판단하는 것은 번거롭습니다. 이 프로그램은 과제명, 마감일, 과제 종류, 난이도를 입력하면 남은 날짜와 중요도를 함께 보여주어 사용자가 오늘 어떤 과제부터 처리해야 하는지 쉽게 결정할 수 있게 도울 것입니다.

## 핵심 기능 3가지
1. 과제명, 과목명, 마감일, 과제 종류, 난이도를 입력해 과제 목록에 추가한다.
2. 현재 날짜를 기준으로 각 과제의 남은 일수를 계산한다.
3. 마감일, 과제 종류, 난이도를 기준으로 과제 목록을 우선순위로 보여준다.

## 예상 사용자
과제와 시험 일정을 관리해야 하는 대학생, 특히 나 자신

## 결과물 형태
CLI 프로그램

## 사용할 Python 개념
변수, 조건문, 반복문, 함수, 리스트, 딕셔너리, 모듈

## 실행 방법
터미널 또는 명령 프롬프트에서 아래 명령어를 입력합니다.

```bash
python main.py
```

프로그램을 실행하면 다음 메뉴가 나타납니다.

```text
1. 과제 추가
2. 전체 과제 보기
3. 우선순위 보기
4. 종료
```

## 파일 구성
- `main.py`: D-day 과제 처리기 프로그램 파일
- `.gitignore`: 필요 없는 자동 생성 파일을 GitHub에 올리지 않기 위한 설정 파일

## 참고
이 프로그램이 실행되는 동안에만 과제 목록을 저장합니다. 프로그램을 종료하면 입력한 과제 목록은 사라집니다.
